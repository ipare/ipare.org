Ipare 文档

渐进式 Node.js 框架，按需组合所需功能

快速上手 GitHub open in new window

渐进式

可以根据项目渐进式安装 Ipare 组件，路由、Ioc、视图、过滤器、请求管道等等。由于核心思想是中间件管道和依赖注入，因此 Ipare 的扩展性更强，你可以按需增加更多其他功能。

多环境运行

Ipare 既可以运行在 serverless 也可以运行在原生nodejs服务、微服务等场景。Ipare 原生支持 serverless，作为云函数比其他nodejs框架速度更快。

先进特性

Ipare 源码全部使用 TypeScript 开发，有完善的智能提示。mva 架构让项目各模块耦合度更低。可选的装饰器功能让接口设计更加简单。
